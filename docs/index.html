Hovercard

Hovercard is a JavaScript library to get Wikipedia cards for terms on mouse over. It's useful in explaining concepts or to give summaries to Wikipedia links.

Code

You can get Hovercard from NPM:

yarn add hovercard

This page imports the Hovercard JavaScript and CSS, and has the following custom CSS:

.hovercard {
  border-bottom: 1px solid;
  color: #69e;
  cursor: help;
}
.hovercard-real {
  box-shadow: 0px 10px 50px rgba(0, 0, 0, 0.2);
  background-color: #fff;
}
.hovercard-arrow {
  border-bottom: 10px solid #fff;
}
.hovercard-real .hovercard-description::after {
  background: linear-gradient(transparent, #fff);
}
